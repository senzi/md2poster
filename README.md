# Markdown to Poster

一个简单易用的Markdown转海报工具，支持实时预览和主题自定义。

## 功能特点

- 支持标准Markdown语法
- 实时预览转换效果
- 自定义主题样式
  - 字体选择（标题/正文）
  - 颜色配置（主题色/背景色/文字颜色）
  - 间距调整（段落间距/行高）
- 响应式布局设计
- 支持多种预设尺寸

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- Vue 3
- TypeScript
- Vite
- CodeMirror 6
- Marked
